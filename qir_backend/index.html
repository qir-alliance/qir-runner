§QIR compliant backend for quantum simulation.

This libary builds on top of the qir_stdlib to implement a full backend for simulation of QIR programs. This includes a broad set of quantum intrinsic operations for sparse state simulation, based on the design from Leveraging state sparsity for more efficient quantum simulations.
